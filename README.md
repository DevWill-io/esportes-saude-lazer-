# 🏆 Projeto Esportes - Site Informativo

Este projeto consiste no desenvolvimento de um **site sobre esportes**, criado com o objetivo de apresentar informações, conteúdos e curiosidades relacionadas ao mundo esportivo, promovendo conhecimento e interação entre os usuários.

O site poderá abordar modalidades esportivas, notícias, atletas, competições, estatísticas e conteúdos educativos voltados ao esporte.

---

## 📌 Objetivos do Projeto

- Desenvolver um site informativo sobre esportes;
- Organizar conteúdos de forma intuitiva e acessível;
- Aplicar conhecimentos de desenvolvimento web;
- Criar uma interface agradável e responsiva;
- Incentivar o interesse pelo esporte e suas modalidades.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** → Estrutura do site  
- **CSS3** → Estilização e design
